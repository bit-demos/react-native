A typography component that defines a standard set of font faces based on Zurich's design system.
## Usage
The classes for using typography styles are the following:
##### Display
<div
  style={{
    display: "grid",
    gridTemplate: "30px 1fr / repeat(2, 1fr)",
    padding: "2rem",
    border: "1px solid var(--color-grayscale-medium-gray)",
    borderRadius: "4px",
    rowGap: "2rem",
  }}
>
  <h3 style={{ margin: 0 }}>Regular</h3>
  <h3 style={{ margin: 0 }}>Decorative</h3>
  <p className="display">Display</p>
  <p className="display-decorative">Display decorative</p>
</div>
<br />
Use the classes `display` or `display-decorative`.  

```js
<p className="display">Display</p>
```  

```js
<p className="display-decorative">Display</p>
```


##### Body
<div
  style={{
    display: "grid",
    gridTemplate: "30px 1fr / repeat(2, 1fr)",
    padding: "2rem",
    border: "1px solid var(--color-grayscale-medium-gray)",
    borderRadius: "4px",
    rowGap: "2rem",
  }}
>
  <h3 style={{ margin: 0 }}>Regular</h3>
  <h3 style={{ margin: 0 }}>Medium</h3>
  <p className="body-lg-r">Body Large Regular</p>
  <p className="body-lg-m">Body Large Medium</p>
  <p className="body-sm-r">Body Small Regular</p>
  <p className="body-sm-m">Body Small Medium</p>
</div>
<br />

The structure of this classes is {`body-\{size}-{weight}`}
**Size**: Large `lg` or small `sm`
**Weight**: Regular `r` or medium `m`

```js
<p className="body-lg-r">Body Large Regular</p>
```

```js
<p className="body-lg-m">Body Large Medium</p>
```

```js
<p className="body-sm-r">Body Small Regular</p>
```

```js
<p className="body-sm-m">Body Small Medium</p>
```
